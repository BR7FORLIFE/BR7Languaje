📌 Resumen General
    Operación	Resultado en la mayoría de lenguajes
    String + int	Concatenación ("25" → "Edad: 25")
    int + int	Suma (10 + 5 = 15)
    int + boolean	Suma (10 + true = 11)
    String + boolean	Concatenación ("Estado: true")
